Data compression is when you shrink the size of data while still conveying (mostly) the same information.

How do we measure data size?

Some examples of digital data are documents, apps, videos, and game saves.

The smallest amount of data we can have is called a bit. A bit represents an on/off decision. One way to think about a bit is like a light bulb, which is either on or off.

While a bit can only have two states, if we compine bits then they can have lots more states. For instance, if we take eight bits, we can have two hundred and fifty six states. We call a group of eight bits a byte.

How was Data Compression first used?

Data Compression has been, and still is, used to minimize the transmition time of information. The earliest example of this is Morse Code, which was used to communicate over primitive and unreliable radios.

Morse code, invented in 1838, is the earliest instance of data compression in that the most common letters in the English language such as “e” and “t” are given shorter Morse codes.
